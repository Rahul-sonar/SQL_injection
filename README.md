# SQL_injection

# Objective
To understand SQL Injection vulnerabilities and perform practical exploitation using SQLMap on a vulnerable web application.

# Tools Used
- SQLMap
- DVWA (Damn Vulnerable Web Application)

# Work Done
- Identified injectable parameter in DVWA
- Used SQLMap to detect SQL Injection
- Extracted database names
- Extracted tables from database
- Dumped user credentials
- Analyzed impact of SQL Injection
- Documented prevention techniques

# Repository Contents
- `Sql_injection_exploitation.pdf`
- (SQLMap outputs and extraction results)
- `README.md`

# Result
demonstrated how SQL Injection vulnerabilities can expose sensitive database data and emphasized the importance of secure input handling.
